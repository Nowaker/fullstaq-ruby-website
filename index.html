What is Fullstaq Ruby?

Fullstaq Ruby is an MRI-based Ruby distribution that's optimized for server production use cases. It is compiled with the Jemalloc and malloc_trim patches, allowing lower memory usage and higher performance.

We supplies binaries (DEBs, RPMs) so that you don't have to compile Ruby from source, and so that you can easily keep Ruby security-patched.

For the community, by the community

We intend for Fullstaq Ruby to be a true community project, welcoming contributors all over the world, and maintainable by anyone. To make this work, we are working on a lot more developer and contributor documentation.

More improvements on the way

Fullstaq Ruby is in beta: it is a work-in-progress. For example, we have DEBs/RPMs, but no APT/YUM repository yet. Further memory reduction optimizations are possible. And much more.

So stay tuned, folks!
